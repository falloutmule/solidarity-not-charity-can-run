# Solidarity Not Charity Can Run

**Solidarity Not Charity Can Run** is a one-file browser game.

**SNC Can Run** is the short name for chat, notes, and quick references. The public title is **Solidarity Not Charity Can Run**.

## Play

- Open `index.html` directly in a browser, or
- Play through GitHub Pages:
  - https://falloutmule.github.io/solidarity-not-charity-can-run/

## Scope

This is intentionally scoped as a single-file static prototype:

- one `index.html` file
- no build step
- no backend
- no online leaderboard
- no external assets
- no external libraries

Saves, stats, and leaderboards are local to the browser using `localStorage`.

Note: browser storage is per-site origin. Saves made from a downloaded `content://` or local file copy will not automatically carry over to the GitHub Pages URL.
