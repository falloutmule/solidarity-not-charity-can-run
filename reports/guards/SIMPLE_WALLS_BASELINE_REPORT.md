# Simple Walls Baseline Report

Goal: reduce SNC Can Run FPV building wall rendering back to a simple technical baseline. `simplewalls1` should draw building module hits as one material color per wall slice, with no active role-dependent bands, decals, panels, facade textures, top/base bands, or framed objects.

BUILD_ID: `simplewalls1`

Previous deployed baseline: `calmwalls1`

## Files changed

- `src/js/game-06-section-2b-mobile-touch-input.js`
  - changed `BUILD_ID` to `simplewalls1`
  - added `CR_SIMPLE_WALLS_BASELINE = 1`
- `src/js/game-09-section-3-city-generation.js`
  - added `crSimpleWallColor(...)`

  - updated the source-build BUILD_ID assertion to `simplewalls1`

## Visual change

- Active module/facade building hits now use `crDrawSimpleWallColumn(...)`.
- The active `simplewalls1` path fills each raycast wall slice with a single material color.
- The active simple path does not call:
  - `crCalmWallBand(...)`
  - `crFacadeArtColBand(...)`
  - `crDrawContinuousFacadeTextureColumn(...)`
  - `crDrawSmoothBuildingFaceColumn(...)`
  - framed box helpers
- Existing facade metadata remains available for compatibility/material category only.
- No gameplay, collision, map generation, props, NPCs, pickups, exit, minimap, mobile controls, save data, or Hall behavior was intentionally changed.
